
# 🛍️ Leiden Deco

Sitio web de ecommerce enfocado en la venta de productos de decoración para el hogar, como velas, velones, difusores, floreros, home spray, jabones y accesorios.

## 📌 Descripción

**Leiden Deco** es una tienda online desarrollada con tecnologías web básicas, pensada para ofrecer una experiencia de navegación simple, moderna y adaptable a distintos dispositivos.

El proyecto incluye:
- Página principal con productos destacados
- Navegación entre secciones
- Carrito de compras (estructura básica)
- Formulario de contacto y newsletter
- Diseño responsive

## 🚀 Tecnologías utilizadas

- HTML5  
- CSS3  
- Font Awesome (iconos)  

## 🎯 Funcionalidades

✅ Navegación principal (Home, Tienda, Contacto)  
✅ Sección de productos destacados
✅ Tienda de productos
✅ Banner publicitario
✅ Diseño responsive (adaptado a mobile y desktop)  
✅ Footer con información de contacto y redes sociales  
✅ Formulario de suscripción a newsletter  
✅ Integración de mapa de ubicación (Google Maps)  

## 📱 Responsive Design

El sitio está diseñado siguiendo el enfoque **desktop-first**, utilizando:

- HTML, CSS, Flexbox y CSS Grid  
- Media Queries  
- Layout adaptable según resolución
